Цели и задачи:

Создание экосистемы обработки данных. Путем консолидации высоконагруженных потоков данных между множеством внутренних и внешних информационных систем. В 16 регионах со своими особенностями технических решений и разными форматами данных.

Ключевые работы:

  • разработка и рефакторинг 190+ интеграционных сервисов для работы с брокером сообщений
  • оптимизация высоконагруженных потоков данных с использованием современных средств разработки highload приложений для соединения между собой 20+ информационных систем
  • профилирование интеграционных сервисов для обеспечения производительности на уровне обработки 10 000+ запросов в секунду
  • организация централизованного и структурированного логирования при помощи трассировки запросов для упрощения анализа потоков данных и качественного контроля ошибок

Результаты:

  • оптимизированы внутренние процессы обработки данных благодаря созданию и совершенствованию единой интеграционной платформы
  • произведено архитектурное проектирование собственного продукта на базе свободного и открытого ПО (OpenSource), которое позволяет 5 млн+ клиентов ежедневно ощущать пользу нашего решения
  • налажено взаимодействие трёх команд разработки в рамках методологии Kanban для увеличения показателя time-to-market заказчика на x0.5
  • выработан единый подход и разработаны типовые архитектурные решения для быстрого создания интеграционных сервисов по готовым шаблонам
