Цели и задачи:

  • создание интеграционного ландшафта крупной энергетической компании с множеством филиалов и участников информационного взаимодействия

Ключевые работы:

  • разработка и рефакторинг 190+ интеграционных сервисов для работы с брокером сообщений
  • оптимизация высоконагруженных потоков данных с использованием современных средств разработки highload приложений для соединения между собой 20+ информационных систем
  • профилирование интеграционных сервисов для обеспечения производительности на уровне обработки 10+ тыс запросов в секунду
  • организация централизованного и структурированного логирования при помощи трассировки запросов для упрощения анализа потоков данных и качественного контроля ошибок

Результаты:

  • оптимизация внутренних процессов обработки данных благодаря созданию и совершенствованию единой интеграционной платформы
  • создание собственного продукта на базе свободного и открытого ПО (OpenSource), которое позволяет 5+ млн клиентов ежедневно ощущать пользу нашего решения
  • увеличение показателя time-to-market заказчика
  • единый подход и типовые архитектурные решения для быстрого создания интеграционных сервисов по готовым шаблонам
