Цели и задачи:

  • автоматизация процесса мониторинга и обработки заявок на грузоперевозки, поступающих из внешних информационных систем
  • обеспечение оперативного поиска ближайшего исполнителя для выполнения заказа

Ключевые работы:

  • создание веб-приложения для получения и обработки заказов на грузоперевозки в режиме реального времени:
    • интеграция с различными Google API (в т.ч. обеспечение построение маршрутов, реализация механизмов отслеживания геопозиции)
    • реестр для ведения данных о машинах, водителях, владельцах
    • аналитические панели для визуализации данных по грузоперевозкам
  • создание мобильного приложения для доступа к данным о заказах на грузоперевозки водителям и владельцам транспортных средств:
    • реализация сбора геоданных о местонахождении водителей
    • реализация механизма пуш-уведомлений для оповещения об изменении статуса грузоперевозки
    • реализация механизма фиксации сведений о процессе грузоперевозки

Результаты:

  • обеспечена стабильная обработка более 300 000 заявок в месяц на грузоперевозки
  • оптимизирован процесс подбора подходящих машин для выполнения заявок на грузоперевозки при наличии более 1 000 машин в реестре
  • реализована интеграция с Document AI для снижения времени ввода информации о документах
